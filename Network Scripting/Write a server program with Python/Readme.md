# Write a server program with Python

Up to this point, we have focused on building a Python client that can connect to a server, receive data, and handle errors gracefully. In this module, we will complete the picture by writing the server-side component of our application.

A server is responsible for listening on a specific IP address and port, accepting incoming connections from clients, and then responding appropriately. Understanding how to implement a basic server in Python is essential for anyone interested in penetration testing, network programming, or building custom security tools.

2. [Testing our Client and Server](https://github.com/vsang181/Get-Good-at-Python/blob/main/Network%20Scripting/Write%20a%20server%20program%20with%20Python/Testing%20our%20Client%20and%20Server.md)
